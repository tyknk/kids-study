# 小3算数 単元マップ

## 使い方
新しい学習アプリを作るときは：
1. 下の表から作りたい単元を選ぶ
2. 「PDFファイル」と「スキャンページ（概算）」を確認
3. 「〇〇の単元でアプリを作って」と伝えるだけでOK

## アプリ作成の手順メモ

```
1. 単元マップで「PDFファイル」と「スキャンページ」を確認
2. pdftoppm でその範囲を画像変換
   例: pdftoppm -png -r 150 -f 3 -l 8 PDF1 /tmp/unit/p
3. 画像を読んで内容・問題パターンを把握
4. CLAUDE.md のデザインガイドラインに沿ってHTML生成
5. 作成済み欄に追記
```
